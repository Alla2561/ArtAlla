  • contact.html (Контакты)

6. Базовый код JavaScript (script.js)

javascript
document.addEventListener('DOMContentLoaded', function() {
    const navLinks = document.querySelectorAll('nav ul li a');

    navLinks.forEach(link => {
        link.addEventListener('click', function(event) {
            event.preventDefault();
            document.querySelector(this.getAttribute('href')).scrollIntoView({
                behavior: 'smooth'
            });
        });
    });
});

7. Шаги по размещению сайта на GitHub Pages

  1. Создайте репозиторий на GitHub.
  2. Загрузите все файлы HTML, CSS и JavaScript в репозиторий.
  3. Перейдите в настройки репозитория.
  4. Прокрутите вниз до секции "GitHub Pages".
  5. В разделе "Source" выберите ветку "main" или "master".
  6. Нажмите "Save".
  7. Ваш сайт будет доступен по ссылке https://<ваше_имя_пользователя>.github.io/<название_репозитория>.

8. Интеграция Google Analytics

  1. Создайте аккаунт в Google Analytics и создайте новый ресурс для вашего сайта.
  2. Скопируйте предоставленный вами код отслеживания.
  3. Вставьте код перед закрывающим тегом </head> в ваших HTML-файлах:
html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=UA-XXXXXXXXX-X"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());

  gtag('config', 'UA-XXXXXXXXX-X');
</script>
<!-- End Google Analytics -->
